## Run locally

You need have Node.js 20 or higher installed.

```bash
  # Clone the repository
  git clone git@github.com:zvolcsey/event-ticket-nextjs.git
  cd event-ticket-nextjs
```

```bash
  # Create the .env from the .env.template
  cp .env.template .env
```
